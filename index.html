Under the Hood

Detection Pipeline

MalGuard processes every file through a 5-stage ensemble pipeline combining classical machine learning, deep learning, and transformer-based NLP — all voting together for maximum accuracy.

01
File Ingestion
Raw bytes extracted, entropy computed, PE headers parsed, file type verified against magic bytes.
02
NLP Tokenization
API call sequences, opcodes, and strings tokenized as natural language using SentencePiece vocabulary.
03
Ensemble Inference
MalBERT, CNN, Bi-LSTM, Random Forest, and GPT-4 all independently score the sample in parallel.
04
Verdict Fusion
Weighted confidence voting aggregates model outputs into a single risk score with 99.1% accuracy.
