# MalGuard
AI-powered malware analysis platform featuring file scanning, URL reputation analysis, hash lookup, code inspection, and an interactive threat intelligence dashboard.

## Features

- File malware scanning
- URL reputation analysis
- Hash lookup
- Code analysis
- Threat intelligence dashboard

> Note: This repository currently demonstrates the frontend interface. Backend malware detection and AI inference are not included.

 ## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## Future Improvements

- Backend malware scanning engine
- VirusTotal API integration
- Real-time sandbox analysis
- Machine learning inference
- User authentication
- Scan history
- Report export (PDF)


## Run

Open `index.html` in a browser.
